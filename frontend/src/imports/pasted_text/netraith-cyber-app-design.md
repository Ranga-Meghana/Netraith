Design a hyper-futuristic, cinematic web application experience for a cybersecurity platform named “Netraith”.

This is not a normal UI. It should feel like a high-end sci-fi interface, similar to a movie-level digital control system.

START WITH A CINEMATIC EXPERIENCE:

Scene 1: Deep Space

* Begin with a zoomed-out view of the universe
* Dark space with glowing stars, nebula clouds, subtle motion
* Smooth gradient transitions (deep black, midnight blue, violet hints)
* Floating particles and slow parallax motion
* The word “NETRAITH” appears with a glowing, minimal futuristic logo

LOGO DESIGN:

* Create a sleek, minimal logo that represents a “network ghost” or “digital wraith”
* Abstract symbol combining:

  * network nodes
  * flowing ghost-like curves
  * sharp cyber lines
* Style: neon cyan + soft glow + thin lines
* Should look like a real cybersecurity product logo

Scene 2: Zoom into Earth

* Camera smoothly zooms from space into Earth
* Earth should glow with cyber activity
* Show animated network connections across continents
* Add red/orange pulses indicating cyber attacks
* Subtle grid overlay to represent digital layer

Scene 3: Transition to Interface

* Seamless morph from Earth into a cyber interface
* Earth dissolves into a futuristic dashboard

DASHBOARD DESIGN:

Style:

* Ultra-modern, glassmorphism + neon cyberpunk
* Dark academia + sci-fi blend
* Smooth gradient background (deep blue → teal → black)
* Floating panels with blur and glow

Layout:

* Top bar with logo “Netraith” and navigation (Dashboard, Threat Map, Logs, Analytics)
* Central area: animated network visualization / threat map
* Floating alert cards:

  * “Threat Detected”
  * IP address
  * Severity level (red glow)
* Side panels:

  * System Status
  * Threat Level
  * Active Alerts
* Bottom control panel:

  * Simulate Attack
  * Scan Network
  * Stop

ANIMATIONS & INTERACTIONS:

* Smooth cinematic transitions (zoom, fade, morph)
* Hover effects:

  * neon glow
  * slight scale-up
  * soft shadow bloom
* Cards:

  * fade-in + slide-up
* Buttons:

  * glowing border + ripple effect
* Background:

  * subtle animated waves or flowing data lines
  * feels alive, never static

COLOR SYSTEM:

* Primary: deep black / midnight blue
* Accent: neon cyan, electric teal
* Alert: red / orange glow
* Text: soft white with slight glow

OVERALL FEEL:

* Extremely futuristic
* Clean but powerful
* Minimal yet cinematic
* Feels like a real cyber defense control system
* Premium, startup-level, visually unforgettable

IMPORTANT:
Ensure the design is structured in sections so it can later be implemented using HTML, CSS, and JavaScript.
